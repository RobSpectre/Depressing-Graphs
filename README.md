## Depressing Graphs

A mess of [IPython notebooks](http://ipython.org/notebook.html) of depressing
graphs.

Check out more at [Depressing Graphs](http://www.depressinggraphs.com).

![Depressing Graphs](https://raw.github.com/RobSpectre/Depressing-Graphs/master/images/depressinggraphs_logo-480x480.png)

## Files

* Notebook files - all end in ipynb.  use 
* requirements.txt - all the dependencies I use to create these graphs.
* /data - local copies of relevant data where not easily accessible.


## Dependencies

Quick list from memory of the core dependencies for these graphs.

* [pandas](http://pandas.pydata.org/) - My favorite data analysis library for
  Python.
* [statsmodels](http://statsmodels.sourceforge.net/) - The best data modeler
  available for Python.
* [vincent](https://github.com/wrobstory/vincent) - Easily translate Python
  datastructures (including pandas DataFrames) into
  [Vega](http://trifacta.github.io/vega/) for d3 graphing.
* [matplotlib](http://matplotlib.org/) - More advanced graphing.
